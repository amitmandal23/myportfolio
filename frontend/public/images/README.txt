Place your website images in this folder.
For example, if you add an image named "hero-1.jpg", you can reference it in your code as "/images/hero-1.jpg".

Recommended images to add:
1. hero-1.jpg (For the first slider slide)
2. hero-2.jpg (For the second slider slide)
3. hero-3.jpg (For the third slider slide)
4. about.jpg (For the About section)
5. logo.png (For the website logo)
